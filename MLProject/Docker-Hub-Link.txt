## MLflow Build-Docker Commands (ADVANCE Criteria)

### 1. Train Model dan Generate MLflow Artifact
```powershell
py -3.12 modelling.py --n_estimators 100 --max_depth 10 --random_state 42
```

**Run ID Generated:** `a648fd18a6cf400296c53ec22a3db8db`  
**Model Registered:** ME_CFS_Depression_Model (version 1)

### 2. Build Docker Image using MLflow
```powershell
py -3.12 -m mlflow models build-docker -m "runs:/a648fd18a6cf400296c53ec22a3db8db/model" -n "junpito/me-cfs-model:latest"
```

**Alternative dengan Registered Model:**
```powershell
py -3.12 -m mlflow models build-docker -m "models:/ME_CFS_Depression_Model/1" -n "junpito/me-cfs-model:latest"
```

### 3. Push to Docker Hub
```powershell
docker login
docker push junpito/me-cfs-model:latest
```

## Alternative: Manual Docker Build

### Build Docker Image
```powershell
docker build -t junpito/me-cfs-model:latest .
```

### Run Docker Container
```powershell
docker run -p 8080:8080 junpito/me-cfs-model:latest
```
